Interactive Story Adventure

This project is an interactive, choice-driven story built using HTML, CSS, and JavaScript. The user takes on the role of Thor and progresses through the story by entering decisions that influence the storyline and outcome. The program dynamically updates the narrative, validates user input, and reveals different story paths based on the choices made. This project demonstrates DOM manipulation, event handling, conditional logic, and interactive user-driven storytelling.

Interactive Car Animation

This project is an interactive car animation built using HTML, CSS, and JavaScript. It visually simulates a car driving along a road, where clicking the car increases its speed and a stop button halts the movement. The program tracks the car’s position and triggers a crash alert when a specific point is reached. This project demonstrates event handling, animation using intervals, DOM manipulation, and basic game-style logic in JavaScript.

Shopping Cart Web Application

This project is a dynamic shopping cart system built using HTML, CSS, and JavaScript. It displays a list of snack products with images, prices, and quantity controls. Users can increase or decrease item quantities, and the program automatically updates individual totals and calculates the final checkout amount. The application demonstrates DOM manipulation, array methods, event handling, and dynamic rendering in JavaScript.

Number Guessing Game

This project is an interactive number guessing game built using HTML, CSS, and JavaScript. The program generates a random number and allows the user to guess within a specified range. It provides instant feedback indicating whether the guess is too high, too low, or correct, and ends the game once the correct number is guessed. This project demonstrates user input handling, conditional logic, DOM manipulation, and basic game flow control in JavaScript.
